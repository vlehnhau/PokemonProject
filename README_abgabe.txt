#### Zur Abgabe ####
####################a

 Bitte stellen Sie sicher, dass alle folgenden Dateien und Ordner zu git hinzugefügt, committed und gepushed sind. Mit dem Befehl git status können Sie sehen, ob Dateien noch nicht hinzugefügt sind. 

Checkliste:	
1) Selbstständigkeitserklärung
2) Source-Code (*.cpp / *.h) inkl. Konsole falls diese verwendet wurde, insbesondere falls diese modifiziert wurde
3) CMakeFiles.txt oder *.pro (nicht die *.pro.user Datei)
3) Ressourcen (Levels,,falls verwendet Bild und Tondateien) im "copyme"-Ordner
4) README.txt mit Quellen,  Systemumgebung, Übersetzung, Bedienung
5) Architektur.txt mit Beschreibung der Architektur

Überprüfen Sie am besten noch einmal über das Webportal von Gitlab ob alle Dateien vorhanden sind. Die Abgabe muss so geschehen, dass sie sich direkt mit einem Build-Tool in der Kommandozeile übersetzen lässt und funktional ist, nachdem der Inhalt des "copyme"-Ordners in das Buildverzeichnis kopiert wurde.
